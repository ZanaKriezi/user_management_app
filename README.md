# User Management App

User Management App - React Internship Challenge

## Features
- Fetch and display users from a REST API
- Search users by name or email
- View user details (address, phone, website)
- Add new users locally with form validation
- Edit and delete users
- Sort users by name or email
- Redux for global state management
- Responsive layout with Material UI

## Tech Stack
React, Redux Toolkit, React Router, Material UI,Fetch API

## Run Locally
npm install
npm run dev
